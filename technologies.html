The Technology



Wix

Wix is the website builder that we have been recommended to use by our client since the Biomotum website has been created with Wix. This is where the graphical web portal will be developed and hosted.

Google Charts

Google Charts is a powerful, simple, and free charting tool that will allow us to create various charts and graphs from the exoskeleton data. We will use this to display live data on our Wix site.

Python

Python will be the language used to retrieve exoskeleton data from Biomotum's AWS server. There will be a simple script that pulls in all of the data and organizes it for simple integration with Google Charts. Lastly a recurring AWS ping will be implemented to keep the data up to date.

JavaScript

JavaScript is the only backend language supported by Wix, therefore we will need to use it in order to connect the hosted Google Charts into our Wix site as well as keep it updated.

GitHub

GitHub will be the repository platform we use to store all of the code for this project.
