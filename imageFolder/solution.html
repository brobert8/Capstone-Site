Our Solution



A Biomotum Web Based Portal

Team Rehab Remote is building a web-portal for Biomotum.

Plan

To cover the requirements from 'The Project' section, we plan on implementing a user account system that will allow individual users to access their own data, as well as admins to access global data. The patients will be able to see their graphical data and download it as either a PDF of the graphics, or the CSV file that the data originally came in before we organize it. For the admin, we will have a heirarchy of data that will involve all global data of the exoskeletons, down to the clinic data summaries, down to the individual patient data.
